# Extension Icons

This directory should contain the extension icons in the following sizes:

- **icon16.png** - 16x16 pixels
- **icon32.png** - 32x32 pixels
- **icon48.png** - 48x48 pixels
- **icon128.png** - 128x128 pixels

## Icon Design Suggestions

The icon should represent the extension's purpose of upscaling manga/manhwa images with AI.

Design ideas:
- An upward arrow with sparkles/stars (representing AI enhancement)
- A manga page with an upscale symbol
- A magnifying glass over manga-style art
- Colors: Purple gradient (#667eea to #764ba2) to match the popup UI

## Creating Icons

You can use tools like:
- **Figma** - Free design tool
- **GIMP** - Free image editor
- **Photoshop** - Professional image editor
- **Online Icon Generators** - Many free options available

## Quick Solution

For testing purposes, you can:
1. Create a simple PNG with text "AI" in each size
2. Use an online favicon generator to create all sizes at once
3. Find free icons from sites like flaticon.com or icons8.com

## File Format

- Format: PNG with transparency
- Color mode: RGBA
- Transparent background recommended
